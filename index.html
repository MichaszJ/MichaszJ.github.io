Featured Projects

Personal Project: Orbit Tool

Web app written in Python using Streamlit Allows for the simulation and plotting of various types of orbits (two-body, three-body, circular-restricted three-body, etc.) by using the Runge-Kutta-Fehlberg ODE solver. Also includes functionality to calculate and plot ground tracks of satellite orbits based on orbital parameters.
