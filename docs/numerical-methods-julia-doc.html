Numerical Methods - Julia Implementation - Usage

Solving ODEs (Initial Value Problem)

include("solve_ode_ivp.jl");

Basic example:

function ode(t, y)
    return -2*y
end;

y_out, t_out = rk_45(ode, 1.0, 0, 2, 0.001);

using Plots
plot(t_out, y_out)

More complex example using coupled first-order ODEs:

# initial conditions
re = 6378.0e3
g0 = 9.807

y1_0 = 6500.0e3
y2_0 = 7.8e3

S0 = [y1_0, y2_0]

# ODE system
function ode2(t, S)
    y1, y2 = S
    return [y2, -(g0 * re^2) / (y1^2)]
end;

# getting results
s_out2, t_out2 = rk_45(ode2, S0, 0, 60*70, 1000);
pos_out = s_out2[1];
vel_out = s_out2[2];


# plotting
pos_plot = plot(t_out2, pos_out, ylabel="Position (m)")
vel_plot = plot(t_out2, vel_out, xlabel="Time (s)", ylabel="Velocity (m/s)")

plot(pos_plot, vel_plot, layout=(2,1), legend=false)
