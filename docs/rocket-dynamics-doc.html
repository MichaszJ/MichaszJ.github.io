Rocket Dynamics - Usage

Rocket Launch Trajectory

Defining rocket parameters, functions for variable physical parameters, and constant physical parameters:

# constants
const launch_mass = 68000.0
const mass_ratio = 7.0
const specific_impulse = 390.0
const thrust = 933.91e3
const mass_flux = thrust / (specific_impulse * 9.80665)
const drag_coeff = 0.5
const frontal_area = pi*5^2 / 4

const g0 = 9.80665
const ρ0 = 1.225
const scale_height = 7.5e3
const r_e = 6378.0e3

# defining rocket
example_rocket = [launch_mass, mass_ratio, thrust, mass_flux, drag_coeff, frontal_area]

# initial conditions
const v0 = 0.0
const γ0 = deg2rad(89.85) # pitchover angle
const x0 = 0.0
const h0 = 0.0
const vD0 = 0.0
const vG0 = 0.0

initial_conditions = [v0, γ0, x0, h0]

# parameter for initiating gravity turn
const h_pitchover = 130.0

# functions for variable physical parameters
density_func(h) = ρ0 * exp(-h / scale_height)
drag_func(density, v) = 0.5 * density * (v^2) * frontal_area * drag_coeff
gravity_func(h) = g0 / (1 + h/r_e)^2
mass_func(t) = launch_mass - mass_flux*t

mass_func (generic function with 1 method)

Getting solutions (testing out different solver methods as well)

tolerance_example = 50.0

s_out_45, t_out_45 = rocket_launch_trajectory(
    example_rocket, 
    initial_conditions, 
    0.0, 
    238.776, 
    density_func,
    drag_func,
    gravity_func,
    mass_func,
    h_pitchover, 
    tolerance=tolerance_example, 
    solver_method="rk_45"
)
h_out_45 = s_out_45[3]
x_out_45 = s_out_45[4]
